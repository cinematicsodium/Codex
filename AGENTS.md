# Repository Instructions

This repository is a Codex skill registry.

- Keep repo-scoped skills in `.agents/skills/<skill-name>/SKILL.md`.
- Keep each skill self-contained: put supporting docs under that skill's `references/` folder.
- Do not add custom registries or install scripts unless Codex's native discovery stops being enough.
- When changing a skill, verify its frontmatter still has `name` and `description`.
